Text Case Converter

About This Text Case Converter

Convert text between common naming conventions used in programming. Automatically detects word boundaries from camelCase, snake_case, kebab-case, spaces, and mixed formats.

Supported Cases

  • camelCase — JavaScript variables, Java methods
  • PascalCase — Classes, React components, C# types
  • snake_case — Python, Ruby, Rust variables
  • UPPER_SNAKE_CASE — Constants in most languages
  • kebab-case — CSS classes, URLs, file names
  • dot.case — Java packages, object properties
  • Title Case — Headings, titles
  • Sentence case — Natural text
  • UPPERCASE / lowercase — Full transforms
  • path/case — File paths, URL segments
