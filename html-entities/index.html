About This HTML Entity Encoder/Decoder

Encode special characters to their HTML entity equivalents, or decode HTML entities back to readable text. Essential for safely embedding content in HTML and preventing XSS vulnerabilities.

Features

  • Three Encoding Modes — Named entities, decimal, or hexadecimal
  • Full Encode Option — Encode all characters, not just special ones
  • Auto-Decode — Handles named, decimal, and hex entities automatically
  • Reference Table — Quick lookup for common HTML entities
  • Privacy — 100% client-side, no data leaves your browser
