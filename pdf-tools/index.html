About This PDF Tool

Merge multiple PDFs into one document or split a PDF into individual pages — all in your browser. Uses the pdf-lib library for reliable, standards-compliant PDF manipulation.

Features

  • Merge PDFs — Combine multiple PDF files in any order
  • Split PDF — Extract all pages or specific page ranges
  • Drag to Reorder — Arrange files before merging
  • No File Size Limit — Limited only by your device's memory
  • 100% Private — Files never leave your browser. No upload, no server.
