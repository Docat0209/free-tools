About This Number Base Converter

Convert numbers between binary (base 2), octal (base 8), decimal (base 10), hexadecimal (base 16), and any custom base from 2 to 36. All computation happens locally in your browser.

Features

  • Standard Bases — Binary, octal, decimal, hexadecimal
  • Custom Base — Any base from 2 to 36
  • Real-Time Conversion — Results update as you type
  • Bit Information — Shows bit count, byte count, and max unsigned value
  • Large Numbers — Uses BigInt for arbitrary precision
  • Privacy — 100% client-side, no data leaves your browser
